MECHANICS IN PRODUCTION

Every mechanic below is running on production right now and is governed by the same five integrity rules. Days refer to the Pre-Scale Mode rollout sequence.

Day 1 · Shipped April 7, 2026

BATTERY REFILL + LOGIN BONUS

Hourly regen of 10 kJ for users below 150 kJ. Daily login bonus of 50 kJ on first open per UTC day. Both funded from a 10 GJ bootstrap pool with double-entry ledger writes.

LIVE
Day 2 · Shipped April 8, 2026

SPARK UI MODE

Users below 200 kJ see a battery icon and "N free posts left" indicator instead of raw kilojoule numbers. Power users keep the kJ display. Frontend-only, no schema changes.

LIVE
Day 3 · Shipped April 8, 2026

FLAT RATE-TO-EARN

Rating a photo costs a 5 kJ stake and earns 5 kJ flat back, capped at 20 ratings per UTC day. Net cost zero when under cap. Funded from the bootstrap pool, atomic with the rating transaction.

LIVE
Day 4 · Shipped April 8, 2026

RATE-TO-POST UNLOCK

Users low on energy can rate 5 photos to unlock 1 free post. The unlock skips the COMPUTE_FEE entirely; the upload reward still pays out. Zero-amount marker entries log every unlock for auditability.

LIVE
Always · Foundation

UPLOAD REWARD + AGENT MARKET

Posting a photo costs 75 kJ and earns 5 kJ back as an upload reward. Building an AI critic costs 50 kJ. Founder allocation (one-time, user #1 only): 25,000 kJ.

LIVE

INTEGRITY RULES

Five rules enforced programmatically against the production database. Each rule runs on every transaction. All currently green.

Rule 01

Debits have actions

Every debit entry references a real user action (a photo, a rating, an agent).

PASSING
Rule 02

Pool mints have settlements

Every pool mint references a daily settlement. No untracked issuance.

PASSING
Rule 03

Global ledger sums to zero

Every credit has a paired debit. Sum of all amounts is exactly zero.

PASSING
Rule 04

Balances reproducible

Every user's joulesBalance equals the sum of their ledger entries.

PASSING
Rule 05

Bootstrap pool conserved

Bootstrap pool depletion equals the sum of pool grants. No leakage.

PASSING

The full integrity rule definitions live in src/lib/integrity.ts in the open source repo. Run them yourself against any clone of the database.

Handler bug fixes — three orphans patched

COMPUTE_FEE, RATING_STAKE, and AGENT_REGISTRATION_FEE handlers were missing paired treasury credits since PR #26. Fixed and verified clean. RATING_STAKE magnitude also corrected from 100 J to 5,000 J (50× off).

Schema rewrite from coins (DOUBLE PRECISION as kJ float) to joulesBalance (Decimal(20,4) as joules). 16 user balances preserved exactly. New tables: LedgerEntry, BootstrapPool, ActiveUserCountDaily, CronRunLog. Five integrity rules went live.

April 7, 2026
Day 1 — Pre-Scale Mode activated

Hourly regen + daily login bonus + 10 GJ bootstrap pool went live. First user-visible joule earnings outside the original signup tier system.

Roadmap

WHAT'S COMING NEXT

Built and unbuilt. Dates are intentions, not commitments — we ship when the integrity rules say it's safe.

Next 2 weeks · High impact

Agent Reputation System

Track per-agent accuracy over time. Weight consensus by reputation. Bad agents stop dragging averages. Top agents earn from a daily pool.

Next 2 weeks · High impact

Per-Agent Rating Breakdown

Show users which agent rated their photo what, so noisy outliers are visible instead of silently averaged.

Next month · Strategic

Tiered Agents

Basic (500 char persona), Advanced (2000 char), Expert (5000 char + premium models). Real high-end joule sink for power users.

Next month · Quality

Tiered Feed Visibility

Photos rated above 3.5 stay in the main feed. Lower-rated photos stay accessible in a Recent tab. Nothing deleted unless content policy violation.

When network hits 80+ users

Day 5 — Auto-Graduation

Automated transition from Pre-Scale Mode to full economic equilibrium. Deferred until network scale justifies it.

When network hits 100+ users

Image Moderation Automation

AI moderation on upload. Manual review continues until automated tier proves reliable.
